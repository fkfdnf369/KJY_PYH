### 🌞 채광 & 통풍
- [ ] 거실 창문 방향 및 햇빛 들어오는 정도
- [ ] 맞은편 동과의 거리 (동간 거리 좁으면 채광 나쁨)
- [ ] 창문 열었을 때 바람 통하는지 (판상형은 주방창-거실창이 마주보는 구조가 환기 유리)
- [ ] 북향/저층이면 곰팡이 흔적 특히 체크
- [ ] 거실 및 각 방 창문 조망 사진으로 남겨두기
- [ ] 저녁 방문 시 다른 집 음식 냄새 올라오는지 확인
- [ ] 화장실에서 담배 냄새 올라오는지 확인 (같은 라인 흡연자 파악)

### 🚪 구조 & 공간
- [ ] 방 개수 및 크기 (실제로 가구 들어가는지 상상해보기)
- [ ] 주방 동선 / 수납 공간
- [ ] 현관 수납 / 드레스룸 유무
- [ ] 베란다 확장 여부 및 상태
- [ ] 화장실 개수 및 크기 (안방 화장실 분리 여부)

### 🔨 마감 & 인테리어
- [ ] 도배 상태 (들뜸, 얼룩, 노르스름한 변색 = 냄새 밴 것)
  - 거실 우물형 천장이면 가장자리 들뜸 특히 확인
  - 도배 교체 필요 시 비용 누가 부담하는지 협의
- [ ] 바닥 상태 (긁힘, 들뜸, 삐걱거림)
  - **매트 깔려 있으면 꼭 걷어달라고 요청** (안 보고 계약하면 후회)
  - 복도를 고개 살짝 기울여서 사이드로 보기 (자국, 파임 발견하기 쉬움)
  - 화분 주변 유심히 보기 (습기로 인한 변색 가능성)
  - 바닥 교체비용은 도배의 몇 배 → 도배보다 더 꼼꼼히 볼 것
- [ ] 싱크대 / 붙박이장 상태 및 연식
- [ ] 욕실 타일 / 줄눈 곰팡이 / 실리콘 상태
- [ ] 창문 / 샷시 상태 (결로 흔적, 밀폐력, 이중창 여부)
- [ ] 도어락 / 현관문 상태
- [ ] 누수 흔적 체크
  - 천장 및 바닥 얼룩
  - 주방 다용도실 벽·천장
  - 안방 베란다 벽·천장
  - 방 창문 샷시 실리콘 색깔 (검게 변했거나 물방울 맺힘 여부)
  - 벽에 붙여 세워둔 가구 뒤편
  - **집 들어갔을 때 퀴퀴한 냄새 나면 주의**

### 💧 수도 & 설비
- [ ] 수압 확인 : 세면기 막고 물 세게 튼 후 변기 내리기 → 수압 얼마나 줄어드는지 확인
- [ ] 배수 확인 : 세면기에 물 가득 채운 후 배수구 열어서 잘 빠지는지 확인
- [ ] 온수 수압 / 온수 나오는 속도 따로 확인
- [ ] 보일러 연식 및 브랜드 (평균 7년 주기 교체 → 교체 시기 반드시 확인)
- [ ] 난방 방식 확인 (개별난방 / 중앙난방 / 지역난방)
  - 개별난방 : 보일러 상태 및 최근 교체 여부
  - 중앙난방 : 난방 파이프 인입 위치 확인
  - 지역난방 : 겨울 난방 상태 직접 물어보기
- [ ] 에어컨 배관 위치 / 실외기 공간
- [ ] 전기 콘센트 위치 및 개수 (가전 배치 고려)
- [ ] 통신 (인터넷, TV) 인입 위치

### 🔇 소음
- [ ] 층간소음 확인은 **오후 6~8시 방문**이 가장 효과적
- [ ] 윗집 세대 구성 파악 (중개인에게 물어보기 — 아이 유무, 자전거/킥보드 개수 등)
- [ ] 위아래 층 발소리 실제로 들어보기 (잠깐 조용히 기다려보기)
- [ ] 도로/주차장/놀이터 소음 (창문 열고 확인)
- [ ] 엘리베이터 / 복도와 가까운 방 위치
- [ ] 쓰레기 분리수거장 위치 및 관리 상태 (냄새·소음 원인)

### 📋 법적 & 권리 관계
- [ ] 등기부등본 열람 (근저당, 가압류/가처분 확인)
- [ ] 실거래가 확인 (국토부 실거래가 공개시스템)
- [ ] 임차인 있을 경우 보증금·월세·만기일 확인
- [ ] 임차인 계속 거주 여부 / 내가 입주할 시기 매도자와 협의
- [ ] 가능하면 세입자에게 직접 집 상태 물어보기 (누수, 결로, 소음 등)
- [ ] 하자보수 협의 가능 여부

## 💬 임장 꿀팁 Q&A

### 💧 수압 제대로 확인하는 법
단순히 수도꼭지 하나만 틀면 안 됨. 동시에 여러 곳 틀어야 실제 수압을 알 수 있어.
1. **변기 물 내리면서 + 세면대 동시에 틀기** → 수압이 확 약해지면 낮은 것
2. **샤워기 헤드 틀어보기** → 가장 체감이 확실함
3. **냉수 말고 온수도 따로 확인** → 온수 수압이 냉수보다 약한 경우도 있음
4. 수압이 약하면 집 내부 배관 문제인지 / 동 전체 문제인지 구분 필요
   - 내부 배관 → 가격 협의 또는 수리 요청 가능
   - 동 전체 문제 (특히 고층) → 개인이 해결 불가. 관리사무소에 미리 물어볼 것

### 📋 매매할 때도 등기부등본이 중요한 이유
매매는 전세보다 안전하지만, 등기부를 확인해야 하는 이유는 따로 있어.
- **근저당**: 잔금날 법무사가 동시 말소 처리하므로 일반적으로는 괜찮음
- **가압류 / 가처분**: 있으면 잔금 처리 자체가 꼬일 수 있어서 위험
- **근저당 금액이 매매가보다 큰 경우**: 이상한 매물 신호
- **전 소유자 법적 분쟁**: 등기부에 흔적 남아있음
- 확인 자체는 1분짜리 — 안 보면 리스크가 너무 큼

### 🚿 욕실 줄눈 & 타일 제대로 보는 법
줄눈 자체가 지저분한 건 입주 시 교체하면 되니까 크게 감점 요인 아님.
진짜 봐야 할 건 **방수층 손상 여부**.

| 상태 | 의미 | 대처 |
|------|------|------|
| 표면 곰팡이, 줄눈 변색 | 단순 오염 | 줄눈 교체로 해결 (소규모) |
| 줄눈 깊숙이 곰팡이 + 퀴퀴한 냄새 | 방수층 손상 가능성 | 욕실 전체 공사 필요 (150~300만원+) |
| 타일 눌렀을 때 "텅텅" 울리는 소리 | 타일 들뜸 확실 | 방수층 공사 신호 |

> **현장 팁**: 타일을 손가락 관절로 살짝 두드려봐. "딱딱" → 정상 / "텅텅" → 들뜬 것
